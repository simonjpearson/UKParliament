Overview

Create a small person manager app - it doesn't have to be complicated! A list of people on the left and an area to the right where a user can edit existing people / add new people

Project

We've setup a template project for you. It includes:

  • Angular frontend embedded in a .net core application (All UI components should be implemented in Angular)
  • .net core web app (A person controller exists for you to create an API)
  • Services assembly (To contain any services you might require)
  • Data assembly (Houses the in-memory database context and a simple person model)
  • Test assembly (For tests!)

Instructions

To complete this exercise please modify this solution and do the following:

  • Extend the person model in the data assembly to include some other properties such as DateOfBirth and Address, as well as anything else you'd like to add
  • Display a list of people already configured in the database on the left of the page. Display people as a simple card or similar.
  • Create an editor for people on the right of the list - a user should be able to edit all the properties (apart from Id!)
  • Add the ability to save existing people / create new people
  • Persist this in the backend in the in-memory database has been configured for you
  • Add the ability to delete a person via a dropdown option on the card
  • Please play consideration to the visual design of the finished product
